# 开发与发布说明

## 开发环境配置

### 微信开发者工具设置
1. 打开微信开发者工具，导入项目（选择本项目根目录）
2. 点击"工具" → "构建 npm"（首次打开或新增依赖后需执行）
3. 右上角"详情" → 本地设置 → 勾选 **"不校验合法域名..."**（开发阶段）

## 验证清单

### 基础功能
- [ ] 输入"中国" → 出现"中"、"国"两个字卡
- [ ] 输入"123abc中国" → 只出现"中"、"国"
- [ ] 输入"中中中" → 只出现一个"中"（去重）
- [ ] 清空输入 → 字卡和动画区消失

### 笔顺动画
- [ ] 点击"中"字卡 → 出现"加载中..."提示
- [ ] 加载完成 → 自动播放笔顺动画
- [ ] 动画结束 → "▶ 重播"按钮变绿色可点击
- [ ] 点击重播 → 动画从头重新播放
- [ ] 快速切换字卡 → 只显示最后一次点击的字的动画（无乱序）

### 错误处理
- [ ] 网络断开时点击字卡 → 显示"暂不支持该字"

## 发布前配置

在 [微信公众平台](https://mp.weixin.qq.com) 完成以下配置：

### 1. 填写 AppID
将 `project.config.json` 中的 `YOUR_APPID_HERE` 替换为实际 AppID。

### 2. 添加 request 合法域名
开发 → 开发管理 → 开发设置 → 服务器域名 → request 合法域名：
```
https://cdn.jsdelivr.net
```

### 3. 构建 npm
发布前确认已执行"工具 → 构建 npm"，`miniprogram_npm/` 目录存在。

## 技术说明

### CDN 配置
本项目使用 `hanzi-writer-miniprogram` 库来提供笔顺动画功能，该库从 jsDelivr CDN 加载汉字笔画数据：
- CDN 地址：`https://cdn.jsdelivr.net/npm/hanzi-writer-data@2.0/`
- 使用场景：用户点击字卡时，动态加载该字的笔画信息并播放动画

**开发时**：在微信开发者工具中勾选"不校验合法域名"可跳过域名检查。

**发布前**：必须在微信公众平台的服务器域名配置中添加 `https://cdn.jsdelivr.net`，否则小程序会因域名检查而无法加载数据。
