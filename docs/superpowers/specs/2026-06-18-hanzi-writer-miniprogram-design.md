# 汉字笔顺小程序 设计文档

**日期**: 2026-06-18  
**项目**: how-to-write 微信小程序  
**目标用户**: 小学生（家长辅助查字）

---

## 需求概述

一款微信小程序，让孩子输入不会写的汉字，即可看到该字的笔顺动画演示。

---

## 技术选型

| 项 | 决策 | 原因 |
|---|---|---|
| 平台 | 微信小程序 | 目标平台 |
| 笔顺动画 | hanziwriter-miniprogram | 开源、免费、动画流畅、小程序版适配完善 |
| 字符数据 | HanziWriter 内置 CDN（jsdelivr） | 无需本地存储，按需加载，~100ms |
| 搜索方式 | 直接输入汉字 | 实现最简，不需要本地字典 |

**CDN 域名白名单**：需在微信公众平台 → 开发设置 → request 合法域名添加 `https://cdn.jsdelivr.net`。

## 核心数据流

1. 用户在搜索框输入汉字（如"中忠终"）
2. 每个字符展示为一个可点击的字卡
3. 点击字卡 → HanziWriter 初始化 canvas，从 CDN 拉取该字笔划数据，自动播放动画
4. 若 CDN 返回 404（字不存在）→ 提示"暂不支持该字"
5. 点击"重播"按钮 → `writer.animateCharacter()` 重新播放

---

## 交互细节

- 输入框无需搜索按钮，实时更新字卡列表（`bindinput` 事件）
- 每次点击新字卡，销毁旧的 HanziWriter 实例，重新初始化
- 动画播放完毕后，"重播"按钮高亮可点
- 字卡最多显示输入的前 10 个字（防止过长输入）
- 非汉字字符（数字、字母、标点）自动过滤

---

## 错误处理

| 情况 | 处理 |
|---|---|
| 网络不可用 | 提示"请检查网络连接" |
| 字不在数据库（404） | 提示"暂不支持该字" |
| 输入为空 | 隐藏字卡区和动画区 |
